2026

NanoGPT and Transformer-based Language Modeling

Period: Spring 2026
Role: Teaching / Presentation Preparation
Level: Undergraduate / Graduate introductory level

This course material introduces the core components of a small-scale GPT-style language model. The topics include tokenization, token IDs, vocabulary mapping, embedding layers, positional encoding, causal masking, multi-head self-attention, feed-forward networks, and the overall training pipeline of NanoGPT.

Main topics:

  • Tokenization and token IDs
  • Vocabulary construction
  • Embedding and positional encoding
  • Causal attention mask
  • Multi-head self-attention
  • Transformer block structure
  • Text generation pipeline

2025

Deep Learning for Computer Vision

Period: 2025
Role: Mentoring / Research Guidance
Level: Undergraduate / Graduate research preparation

This mentoring activity focuses on helping students understand how deep learning models are applied to computer vision tasks. The emphasis is on object detection, image segmentation, training pipelines, loss functions, and model evaluation.

Main topics:

  • CNN-based visual feature extraction
  • Object detection with YOLO-style models
  • Image segmentation models
  • Training and validation workflow
  • Evaluation metrics for detection and segmentation
  • Reading and reproducing research papers

Research Mentoring in Boundary-aware Segmentation

Period: 2025
Role: Research Mentoring
Level: Graduate research level

This mentoring topic introduces students to boundary-aware learning and evaluation in image segmentation. The main focus is on understanding why region-overlap metrics such as IoU and Dice may not fully capture structural prediction errors, especially near object boundaries.

Main topics:

  • IoU and Dice-based evaluation
  • Boundary mismatch and structural error
  • False-positive and false-negative decomposition
  • Boundary-aware metrics
  • Camouflaged object segmentation
  • Experimental analysis and result visualization

2024

Python and PyTorch for Deep Learning

Period: 2024
Role: Tutorial / Mentoring
Level: Beginner to intermediate

This course material provides practical guidance for implementing deep learning models using Python and PyTorch. The goal is to help students build a complete training pipeline from dataset loading to model evaluation.

Main topics:

  • Python programming for AI
  • PyTorch tensors and autograd
  • Dataset and DataLoader design
  • Training loop implementation
  • Loss computation and optimization
  • Model checkpointing
  • Result logging and visualization
