2024

Technical Mentoring for Exchange Student

Student background: Japanese exchange student from Ritsumeikan University Period: 2024 Role: Technical mentor Topics: Micro:bit programming, basic embedded systems, introductory AI/IoT practice

I mentored a Japanese exchange student from Ritsumeikan University on a hands-on technical project using the micro:bit platform. The mentoring focused on introducing basic programming, sensor-based interaction, and simple embedded-system concepts through practical exercises.

The micro:bit is a small programmable board designed for educational computing. It allows students to learn fundamental concepts such as input/output control, sensor reading, event-based programming, and simple communication between devices. Through this activity, the student practiced how software instructions can interact with physical hardware in a simple and intuitive way.

Main activities:

  • Introduced the micro:bit platform and its basic components
  • Guided basic programming for input, output, and sensor interaction
  • Supported hands-on exercises using LEDs, buttons, and built-in sensors
  • Explained simple embedded-system and IoT concepts
  • Discussed how programming logic can be connected to real-world physical devices

Course and Tutorial Materials

NanoGPT and Transformer-based Language Modeling

Period: 2026 Type: Course/tutorial material Level: Undergraduate / graduate introductory level

This material introduces the core components of a small-scale GPT-style language model. It explains how text is converted into token IDs, how embeddings are constructed, and how transformer blocks process sequential data using masked self-attention.

Main topics:

  • Tokenization and token IDs
  • Vocabulary construction
  • Embedding and positional encoding
  • Causal attention mask
  • Multi-head self-attention
  • Transformer block structure
  • Text generation pipeline

Deep Learning for Computer Vision

Period: 2025 Type: Course/tutorial material Level: Undergraduate / graduate introductory level

This material introduces practical deep learning concepts for computer vision. The focus is on how visual data are processed, how neural networks are trained, and how model performance is evaluated in detection and segmentation tasks.

Main topics:

  • CNN-based visual feature extraction
  • Object detection with YOLO-style models
  • Image segmentation models
  • Training and validation workflow
  • Loss functions for visual recognition
  • Evaluation metrics for detection and segmentation
  • Reading and reproducing research papers

Boundary-aware Segmentation and Evaluation

Period: 2025 Type: Research-oriented tutorial material Level: Graduate research level

This material introduces boundary-aware learning and evaluation for image segmentation. The main focus is on understanding why region-overlap metrics such as IoU and Dice may not fully capture structural prediction errors, especially around object boundaries.

Main topics:

  • IoU and Dice-based evaluation
  • Boundary mismatch and structural error
  • False-positive and false-negative decomposition
  • Boundary-aware metrics
  • Camouflaged object segmentation
  • Experimental analysis and result visualization

Python and PyTorch for Deep Learning

Period: 2024 Type: Practical tutorial material Level: Beginner to intermediate

This material provides practical guidance for implementing deep learning models using Python and PyTorch. The goal is to help students build a complete training pipeline from dataset loading to model evaluation.

Main topics:

  • Python programming for AI
  • PyTorch tensors and autograd
  • Dataset and DataLoader design
  • Training loop implementation
  • Loss computation and optimization
  • Model checkpointing
  • Result logging and visualization
